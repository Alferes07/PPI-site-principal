Sistemas Para Internet

Eu sou Designer

Sobre o curso:

O tecnólogo em sistemas para internet é responsável por montar sites para internet e aplicativos. Ele também se encarrega da manutenção e soluciona todos os problemas relacionados à tecnologia web.

Detalhes do curso:

  • Áreas de Atuação: Desenvolvimento de Sistemas, Criação e Instalação de Servidores, Coordenação de Projetos, etc...
  • Tempo de duração: 2 - 3 anos
  • Modalidades Disponíveis: Presencial e EaD
  • Tipo: Tecnólogo
  • Média salarial: R$4.846 - >R$10.000

A média salarial pode variar de acordo com o nível de conhecimento (júnior, pleno, sênior, etc...), empresa, função e muito mais.

Habilidades:

No curso de Sistemas para Internet você aprenderá várias linguagens de programação, desde o básico de HMTL, até o avançado de JavaScript.

JavaScript 100%
HTML 90%
CSS 90%
FrameWorks (Node, Bootstrap, etc...) 75%
Banco de Dados 70%
PHP 60%

Faculdades com cursos presenciais

Logo abaixo está listado algumas das principais faculdades próximas à Cristalina que oferecem o curso de Sistemas para Internet.

IFTM - Campus Uberlândia Centro

Possui boas disciplinas e fica em uma cidade grande, onde há mais possíbilidade de conseguir uma moradia melhor. (Residência estudantil apenas depois de um ano)

Tem a melhor matriz curricular, que abrange principalmente as disciplinas de desenvolviomento WEB Front-End. (Possui o programa de residência estudantil.)

IFB - Campus Brasília

A faculdade com uma das melhores e mais bonitas infraestruturas que já vi, sendo também a mais próxima das três opções. (Não possui o programa de residência estudantil.)

Nota de Corte: 739.01 (AC 2020)
